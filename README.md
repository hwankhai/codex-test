- [ ] 기본 웹페이지 구성

- [ ] API 연동 테스트
- [ ] 데이터베이스 연동 테스트

## 사용 방법
1. 프로젝트 클론
```bash
git clone <repository-url>
cd codex-test
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm start
```
